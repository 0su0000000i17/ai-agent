это desktop AI-agent на Go с собственным runtime для управления ПК через естественный язык.

Проект строится как универсальное “умное ядро” для взаимодействия с компьютером: агент анализирует состояние системы, понимает контекст интерфейса и выполняет многошаговые действия в браузере и приложениях без жёсткого хардкода под конкретные сайты.

Основная идея проекта

Вместо набора отдельных скриптов под каждый сервис, развивается как универсальная agent-система:

наблюдение за состоянием ПК
работа с окнами и браузером
анализ интерфейсов
planning / goal loop
выполнение цепочек действий
память и контекст задач

Цель проекта, приблизить поведение агента к “человеку за компьютером”, который сам ориентируется в интерфейсах и выполняет задачи.
